## Requirements
- Node.js installed

## Steps

1. Install the Project Dependencies
```bash
npm i
```
2. Create a .env File
```env.local
NEXT_PUBLIC_API_URL=http://localhost:5000
NEXT_PUBLIC_OPENCAGE_API_KEY=
NEXT_PUBLIC_IMGBB_API_KEY=
```
3. Start the Server
```bash
npm run dev
```
