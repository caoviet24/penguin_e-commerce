## Requirements
- Node.js installed

## Steps

1. Install the Project Dependencies
```bash
npm i
```
2. Create a .env File
```env
NEXT_PUBLIC_API_URL=http://localhost:8080/api/v1
REACT_APP_OAUTH_CLIENT_ID=
REACT_APP_OPENCAGE_API_KEY=
```
3. Start the Server
```bash
npm run dev
```
